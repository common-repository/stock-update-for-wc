Bulk update stock values on your woocommerce store in just few seconds.


== Description ==

Don't waste time in uploading the stock/inventory values on store manually. This plugin enables you store manager
to update the stock values in a matter of seconds.

* Want to update just few products in less interval? Enter SKU and stock value and that's it.

* You will send an email notification each time the stock gets updated. It will reduce the possiblity of any errors.

* Inventory backups will be stored on your server, so that you won't loose any data.
